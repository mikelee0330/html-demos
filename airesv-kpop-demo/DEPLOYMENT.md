## M1 更新方式

1. Clone `mikelee0330/html-demos`。
2. 修改 repository 內的 `airesv-kpop-demo/`。
3. 本機用 `python3 -m http.server` 檢查。
4. Commit 並 push 到 `main`。
5. GitHub Pages 會自動重新發布，網址不變。

本機主要發布包位於：

`airesv_kpop/EX/airesv-kpop-demo/`
